# 🧺 Laundry Data Management App
Aplikasi Android berbasis Kotlin yang digunakan untuk melakukan pendataan pelanggan laundry secara sederhana. Aplikasi ini membantu proses pencatatan, pencarian, dan pengelolaan data pelanggan laundry agar lebih terorganisir dibandingkan pencatatan manual.
## 📱 Deskripsi
Al Fath Laundry App merupakan aplikasi frontend Android yang dikembangkan menggunakan Kotlin. Aplikasi ini memungkinkan pengguna untuk menyimpan data pelanggan laundry, mencari data yang sudah terdaftar, memverifikasi keberadaan data pelanggan, serta menghapus data yang tidak diperlukan atau salah input.
## ✨ Fitur
* ➕ Menambahkan data pelanggan laundry
* 🔍 Mencari data pelanggan
* ✅ Memeriksa apakah data pelanggan sudah terdaftar
* 🗑️ Menghapus data pelanggan
* 📋 Menampilkan daftar data pelanggan
## 🛠️ Teknologi yang Digunakan
* Kotlin
* Android Studio
* Android SDK
* XML Layout
* Material Design Components
## 🚀 Cara Menjalankan
### Prasyarat
* Android Studio
* Android SDK
* JDK yang sesuai dengan konfigurasi proyek
### Instalasi
1. Clone repository
```bash
git clone https://github.com/username/laundry-data-management.git
```
2. Buka proyek menggunakan Android Studio.
3. Tunggu proses Gradle Sync selesai.
4. Jalankan aplikasi pada emulator atau perangkat Android.
## 🎯 Tujuan Proyek
Proyek ini dibuat untuk mempelajari pengembangan aplikasi Android menggunakan Kotlin, khususnya dalam implementasi operasi dasar pengelolaan data seperti:
* Create (Tambah Data)
* Read/Search (Mencari Data)
* Validation (Pengecekan Data)
* Delete (Hapus Data)
